# rofi-ttv

# Dependencies:

Hard coded:
 * `curl`
 * `jq`

Soft coded:
 * `rofi`
 * `youtube-dl`
 * `mpv`

# Configuration:

Don't use `rofi`, `youtube-dl` or `mpv`? No problem, just adjust the `menu` and `viewer` functions (and the dependency check) in the `rofi-ttv` script to you use your prefered applications.
